WP Post Contributors plugin allows you to add more than one author to the post who have contributed.

== Description ==
WP Post Contributors plugin allows you to add more than one author to the post who have contributed.
The contributors meta will apear just aside of your post body. Just check the authors who have contibuted.
And update the post to save it.

Features of Custom News Widget:

* User friendly and easy to use.
* Tested with other popular plugins.
* Add authors to the post easily.

== Installation ==

1. Upload the `wp-post-contributors` files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the `wp-post-contributors` through the 'Plugins' screen in WordPress
3. The contributors meta will apear just aside of your post body.
4. Just check the authors who have contibuted. And publish/update the post to save it.

That's All
